subclassDShelper1 {dsBase}R Documentation

generates subsets vectors from a factor vector

Description

This is an internal function called by the function 'subclassDS'.

Usage

subclassDShelper1(xvect = NULL, xname = NULL, filter = NULL)

Arguments

xvect

a vector of type factor.

nfilter

the minimum number observation (i.e. rows) that are allowed.

Details

The function generates subsets if the input of 'subclassDS' is a factor vector.

Value

a list which contains the subsets.

Author(s)

Gaye, A.
