asListDS {dsBase}R Documentation

Converts a data.frame or matrix into a list

Description

this function is similar to R function 'as.list'

Usage

asListDS(input)

Arguments

input

the object to turn into list.

Details

Unlike the R function 'as.list' some restriction are applied.

Value

a valid list or an empty list
