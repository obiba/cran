corTestDS {dsBase}R Documentation

Tests for correlation between paired samples

Description

This function is similar to R function cor.test.

Usage

corTestDS(x, y)

Arguments

x

a character string providing the name of a numerical vector.

y

a character string providing the name of a numerical vector.

Details

The function runs a two-sided Pearson test with a 0.95 confidence level.

Value

the results of the Pearson test.
