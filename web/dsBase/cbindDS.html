cbindDS {dsBase}R Documentation

Combines objects by columns

Description

This function is similar to the R base function 'cbind'.

Usage

cbindDS(objs = NULL, clnames = NULL)

Arguments

objs

a list which contains the names of the objects to combine.

clnames

a list of charcaters, the column names of the output data frame.

Details

Unlike the R base function 'cbind' this function' output is a dataframe as that is the type mainly used in DataSHIELD.

Value

a dataframe
