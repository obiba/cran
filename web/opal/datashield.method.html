datashield.method {opal}R Documentation

Get Datashield method by its name

Description

Get a Datashield method of a given type by its name.

Usage

datashield.method(opals, name, type = "aggregate")

## S3 method for class 'opal'
datashield.method(opal, name, type = "aggregate")

## S3 method for class 'list'
datashield.method(opals, name, type = "aggregate")

Arguments

opals

Opal object or list of opal objects.

name

Name of the method

type

Type of the method: "aggregate" (default) or "assign".
