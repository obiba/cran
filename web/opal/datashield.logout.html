datashield.logout {opal}R Documentation

Logout from Opal(s)

Description

Clear the Datashield R sessions and logout from Opal(s).

Usage

datashield.logout(opals, save = NULL)

Arguments

opals

Opal object or a list of opals.

save

Save datashield sessions on each opal with provided ID (must be a character string).

[Package opal version 2.6.1 ]
