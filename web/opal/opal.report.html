opal.report {opal}R Documentation

Opal report

Description

Helper function for generating reports.

Usage

opal.report(input, output = NULL,
  boot_style = getOption("opal.report.style"), progress = FALSE,
  verbose = FALSE)

Arguments

input

Path to the R markdown report file

boot_style

The Bootstrap style to use if character, if NULL uses the default markdown to HTML converter (no Bootstrap style), if TRUE a menu is shown with the available styles.

progress

Knitr progress option

verbose

Knitr verbose option

ouput

Directory path where to ouput the html report file. Default is the current working directory.
