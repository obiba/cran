opal.workspace_rm {opal}R Documentation

Remove a R workspace from a opal.

Description

Remove a R workspace from a opal.

Usage

opal.workspace_rm(opal, ws, user = NULL)

Arguments

opal

Opal object.

ws

The workspace name

user

The user name associated to the worskpace. If not provided, the current user is applied.
