table2dhelper3 {dsBaseClient}R Documentation

Generates percents from counts returned by the function table2d.ds

Description

This is an INTERNAL function.

Usage

table2dhelper3(input, valid.output.array, zero.studies.valid,
  unique.categories.1, unique.categories.2, num.cats.1, num.cats.2,
  opals.valid.id, num.valid.tables)

Arguments

input

a list object returned by the server side function table2dDS

valid.output.array

a valid array

zero.studies.valid

a boolean indicating if all studies were not valid

unique.categories.1

a vector, the full set of unique categories across all studies (from valid binaries)

unique.categories.2

a vector, the full set of unique categories across all studies (from valid binaries)

num.cats.1

length of the vector 'unique.categories.1'

num.cats.2

length of the vector 'unique.categories.2'

opals.valid.id

a vector that gives the indices of the studies with valid tabulated vectors

num.valid.tables

number of studies with valid tabulated vectors (no cell with count > 0 and < 5)

Details

The function is required required by the client function ds.table2D.

Value

a list that contains summary tables with columns and rows percents

Author(s)

Burton, P.; Gaye, A.
