table1dhelper2 {dsBaseClient}R Documentation

Combines output of server side function table1d.ds

Description

this is an INTERNAL function required by the client function ds.table1D.

Usage

table1dhelper2(input)

Arguments

input

a list object returned by the server side function table1d.ds

Details

It gathers the objects returned by table1dDS

Value

a matrix containing study specific and overall (combined) tables of counts based ONLY on data from studies where all table categories are VALID.

Author(s)

Burton, P.; Gaye, A.
