table2dhelper4 {dsBaseClient}R Documentation

Finalizes the output of the server side function table2dDS

Description

This is an INTERNAL function.

Usage

table2dhelper4(server.func.output, var.name.1, var.name.2)

Arguments

server.func.output

a list object returned by the server side function table2dDS

var.name.1

a character string, the name of the first input numerical vector

var.name.2

a character string, the name of the second input numerical vector

Details

The function is only called by the client function ds.table2d. It calls other internal functions namely table2dhelper1, table2dhelper2 and table2dhelper3.

Value

a list that contains the elements returned by the client function ds.table2D

Author(s)

Burton, P.; Gaye, A.
