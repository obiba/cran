ds.names {dsBaseClient}R Documentation

Gets the names of a server-side list

Description

Function to get the names of an object that is stored on the server-side.

Usage

ds.names(x = NULL, datasources = NULL)

Arguments

x

a character string specifying the name of the list.

datasources

a list of DSConnection-class objects obtained after login. If the datasources argument is not specified the default set of connections will be used: see datashield.connections_default.

Details

This function is similar to the R function names. In DataSHIELD the use of this function is restricted to objects of type list.

Server function called: namesDS

Value

ds.names returns to the client-side the names of a list stored on the server-side.

Author(s)

DataSHIELD Development Team

  #Get the names of the list
  
  ds.names(x = "D.list",
           datasources = connections)
 
  
  # clear the Datashield R sessions and logout
  datashield.logout(connections)
