ds.corTest {dsBaseClient}R Documentation

Tests for correlation between paired samples in the server-side

Description

This is similar to the R base function cor.test.

Usage

ds.corTest(x = NULL, y = NULL, datasources = NULL)

Arguments

x

a character string providing the name of a numerical vector.

y

a character string providing the name of a numerical vector.

datasources

a list of DSConnection-class objects obtained after login. If the datasources argument is not specified the default set of connections will be used: see datashield.connections_default.

Details

Runs a two-sided Pearson test with a 0.95 confidence level.

Server function called: cor.test

Value

ds.corTest returns to the client-side the results of the Pearson test.

Author(s)

DataSHIELD Development Team
