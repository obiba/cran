table1dhelper3 {dsBaseClient}R Documentation

Generates percents from counts returned by the function table1dDS

Description

This is an INTERNAL function required by the helper function table1dhelper4.

Usage

table1dhelper3(a.boolean, a.matrix, a.list)

Arguments

a.boolean

a boolean that indicates whether or not all the studies are invalid

a.matrix

a matrix containing study specific and overall (combined) tables of counts based ONLY on data from studies where all table categories are VALID - this matrix is generated by the function table1dhelper2

a.list

a list of primary outputs from ALL studies ("STUDY.NAME","IS.STUDY.DATA.VALID?","TABLE.MASKED.COUNTS")

Details

It generates percents from the count tables.

Value

a list that contains summary tables with columns and rows percents

Author(s)

Burton, P.; Gaye, A.
