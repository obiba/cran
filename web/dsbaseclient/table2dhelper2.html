table2dhelper2 {dsbaseclient}R Documentation

Generates a valid array

Description

this is an INTERNAL function required by the client function ds.table2d. The function is not 'exported' and hence not visible to user and is only called by ds.table2d

Usage

  table2dhelper2(input, num.valid.tables,
    opals.valid.binary, opals.valid.id, zero.studies.valid)

Arguments

input

a list object returned by the server side function table2d.ds

num.valid.tables

number of studies with valid tabulated vectors (no cell with count > 0 and < 5)

opals.valid.binary

vector of binaries indicating whether or not the tabulate vector was valid or not (0 = not valid and 1 = not valid)

opals.valid.id

a vector that gives the indices of the studies with valid tabulated vectors

zero.studies.valid

a boolean indicating if all studies were not valid

Value

a list which contains a valid array and vectors of unique categories
