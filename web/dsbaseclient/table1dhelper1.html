table1dhelper1 {dsbaseclient}R Documentation

Checks validity of tabulated factor variable

Description

this is an INTERNAL function required by the client function ds.table1d. It verifies the cell counts of the tabulate vectors which are considered valid if no count > 0 and < 5 and not valid otherwise.

Usage

table1dhelper1(input)

Arguments

input

a list object returned by the server side function table1d.ds

Details

the function is not 'exported' and hence not visible to user and is only called by ds.table1d

Value

a list that contains the following elements: num.opals the number of studies to analyse num.valid.tables number of studies with valid tabulated vectors (no cell with count > 0 and < 5) opals.valid.binary vector of binaries indicating whether or not the tabulate vector was valid or not (0 = not valid and 1 = not valid) opals.valid.id a vector that gives the indices of the studies with valid tabulated vectors zero.studies.valid a boolean indicating if all studies were not valid, 'TRUE' if none of the studies is valid, 'FALSE' otherwise
