glmhelper1 {dsModellingClient}R Documentation

Extracts the elements of an expression of type call

Description

This is an internal function required by the client function ds.glm to extract variables from a regression formula

Usage

glmhelper1(input)

Arguments

input

a call of the form 'x1+x2+...' or 'x1*x2*...', typically this is the rightern side of a regression formula (i.e. the linear predictor) with the intercept.

Value

a list which contains the individual elements of the input expression

Author(s)

Gaye, A.
