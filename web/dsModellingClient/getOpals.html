getOpals {dsModellingClient}R Documentation

Gets the opal objects

Description

This is an internal function.

Usage

getOpals()

Details

The function searches for a list containing object of type 'opal' in the global environment; if more than one list is found it return the lastest. This way no matter what the user calls his opal login object it will be captured.

Value

a list of opal object obtained after login into the servers
