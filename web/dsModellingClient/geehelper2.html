geehelper2 {dsModellingClient}R Documentation

Produces the pooled beta values and standard errors

Description

This is an internal function required by the client function ds.gee.

Usage

geehelper2(score, infoMatrix, Jmatrix, startCoeff)

Arguments

score

a numeric vector, a score vector

infoMatrix

a matrix, the information matrix

Jmatrix

a matrix, the J matrix

startCoeff

a numeric vector, the starting values for the beta coefficients.

Value

a list which contains the individual elements of the input expression

Author(s)

Gaye, A.; Jones EM.
