lexisHelper1 {dsModellingClient}R Documentation

Generates a default value to use as interval width

Description

This is an internal function required by the function ds.lexis

Usage

lexisHelper1(datasources, exitTime)

Arguments

datasources

a list of opal object(s) obtained after login to opal servers; these objects also hold the data assigned to R, as a data frame, from opal datasources

exitTime

a character, the name of the column that holds the exit times.

Details

If the analyst has not provided an interval width used to generate the expanded table in a piecewise exponential regression, this functions generates a default value which is 1/10 of the global mean of the exit time values.

Value

a numeric value
