dsadmin.package_descriptions {opaladmin}R Documentation

Get Datashield Package Descriptions

Description

Get Datashield package descriptions from Opal(s).

Usage

  dsadmin.package_descriptions(opal, fields = NULL)

Arguments

opal

Opal object or list of opal objects.

fields

A character vector giving the fields to extract from each package's DESCRIPTION file in addition to the default ones, or NULL (default). Unavailable fields result in NA values.

Value

A named list of package descriptions

[Package opaladmin version 1.18 ]
