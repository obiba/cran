oadmin.package_description {opaladmin}R Documentation

Get Package Descriptions

Description

Get package description from Opal(s).

Usage

  oadmin.package_description(opal, pkg, fields = NULL)

Arguments

opal

Opal object or list of opal objects.

pkg

Package name.

fields

A character vector giving the fields to extract from each package's DESCRIPTION file in addition to the default ones, or NULL (default). Unavailable fields result in NA values.
