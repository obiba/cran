dsadmin.rm_package_methods {opaladmin}R Documentation

Remove Package Datashield Methods

Description

Remove Datashield aggregate and assign methods defined by the package.

Usage

  dsadmin.rm_package_methods(opal, pkg, type = NULL)

Arguments

opal

Opal object or list of opal objects.

pkg

Package name.

type

Type of the method: "aggregate" or "assign". Default is NULL (=all type of methods).

[Package opaladmin version 1.18 ]
