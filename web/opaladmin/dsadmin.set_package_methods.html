dsadmin.set_package_methods {opaladmin}R Documentation

Set Package Datashield Methods

Description

Declare Datashield aggregate and assign methods as defined by the package.

Usage

dsadmin.set_package_methods(opal, pkg, type = NULL)

Arguments

opal

Opal object or list of opal objects.

pkg

Package name.

type

Type of the method: "aggregate" or "assign". Default is NULL (=all type of methods).

Value

TRUE if successfull

[Package opaladmin version 1.20.0 ]
