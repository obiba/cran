recodeValuesDS2

Description

The second serverside function called by ds.recodeValues.o This is an aggregate function.

Usage

recodeValuesDS2.o(var.name.text = NULL, values2replace.text = NULL,
  new.values.text = NULL, numeric.output.format.possible,
  force.output.format = "no", v2r.numeric = NULL)

Arguments

var.name.text

a clientside specified character string identifying the variable to be recoded

values2replace.text

a clientside specified character string identifying the original values to be changed

new.values.text

a clientside specified character string identifying the new values to be allocated the values identified by values2replace.text and new.values.text are of the same length and in the same order

numeric.output.format.possible

a clientside generated Boolean variable indicating whether the vector of new values contains any non-numerics which will make a solely numeric output impossible

force.output.format

a clientside provided character string determining whether to force a numeric output. If so, and if the the vector of new values contains non-numerics they will be set to NaN

v2r.numeric

a clientside provided Boolean which if TRUE forces all input values (values2replace) to be numeric. If so, and if the the vector of values2replace contains non-numerics they will be set to NaN

Details

For more details see the extensive header for ds.recodeValues.o
