recodeValuesDS1

Description

The first serverside function called by ds.recodeValues.o This is an aggregate function.

Usage

recodeValuesDS1.o(var.name.text = NULL, values2replace.text = NULL,
  new.values.text = NULL)

Arguments

var.name.text

a clientside specified character string identifying the variable to be recoded

values2replace.text

a clientside specified character string identifying the original values to be changed

new.values.text

a clientside specified character string identifying the new values to be allocated the values identified by values2replace.text and new.values.text are of the same length and in the same order

For more details see the extensive header for ds.recodeValues.o
