glmDS2 {dsModelling}R Documentation

Fits a generalized linear model

Description

Fits a GLM for one iteration only

Usage

glmDS2(formula, family, beta.vect, offset, weights, data)

Arguments

formula

an object of class formula.

family

a character, description of the error distribution and link function to used in the model.

beta.vect

a string character: the starting values for the parameters in the linear predictor.

offset

a character, null or a numeric vector that can be used to specify an a priori known component to be included in the linear predictor during fitting.

weights

a character, the name of an optional vector of 'prior weights' to be used in the fitting process. Should be NULL or a numeric vector.

data

a character, the name of an optional data frame containing the variables in in the formula.

Details

This function cannot be called through 'datashield.aggregate' like other aggregate functions It is only called from within the client function ds.glm in the package dsModellingClient

Value

a list which contains: the fitted family, a score vector and an information matrix

Author(s)
