lexishelper2 {dsModelling}R Documentation

Computes time periods of each subject

Description

This is an internal function required by the client. function ds.lexis to get the time periods for each subject.

Usage

lexishelper2(dt, ids, exits, interval)

Arguments

dt

the dataset that contains the original data.

ids

a string character, the name of column that holds the individuals IDs.

exists,

a string character, the name of column that holds the exit times.

interval

a numeric vector, the sought interval widths.

Details

If the exit time information is missing one single break is returned with the a missing value.

Value

a list which contains the individuals time periods.
