glmDS {dsModelling}R Documentation

Fits a generalized linear model

Description

Fits a GLM for one iteration only

Usage

glmDS(formula, family, beta.vect = NULL)

Arguments

formula

an object of class formula

family

a description of the error distribution and link function to used in the model

beta.vect

a string character: the starting values for the parameters in the linear predictor

Details

This function cannot be called through 'datashield.aggregate' like other aggregate functions It is only called from within the client function ds.glm in the package dsmodellingclient

Value

a list which contains: the fitted family, a score vector and an information matrix

Author(s)

Burton, P.; Laflamme, P.; Gaye, A.
