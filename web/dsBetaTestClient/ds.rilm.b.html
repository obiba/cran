Random Intercept Linear Model (RILM)

Description

Runs a Random Intercept Linear Model on horizontally-partinioned data

Usage

ds.rilm.b(..., y = NULL, datasources = NULL)

Arguments

...

the explanatory variables, could be factor or numeric vectors

y

the dependent variable

datasources

a list of opal object(s) obtained after login to opal servers; these objects also hold the data assigned to R, as a dataframe, from opal datasources.

Details

see description

Value

regression outcome
