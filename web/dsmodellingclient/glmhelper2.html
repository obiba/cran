glmhelper2 {dsmodellingclient}R Documentation

Extracts object variables from a regression formula

Description

Extracts object variables from a regression formula

Usage

  glmhelper2(input)

Arguments

input

a regression formula of the form 'y~x2+x2...' or 'y~x1*x2+x3...'

Details

This is an internal function required by the client function ds.glm to extract variables from a regression formula

Value

a list which contains the individual objects of the linear predictor but not the 'intercept' if there is one.

Author(s)

Gaye, A.
